# AVR_-Washing_Machine
AVR_ Washing_Machine


💡 수행목표

- ATmega128A MCU에 대한 이해 및 C 프로그래밍
- Tact Swich를 이용한 PMW DC Motor Speed Control
- 초음파 거리센서 모듈을 이용한 동작 멈춤 구현
- 실제 세탁기 동작과 유사한 auto 동작 구현
- 동작 과정 및 시간을 LCD에 표시
- 타이머 인터럽트의 이해
